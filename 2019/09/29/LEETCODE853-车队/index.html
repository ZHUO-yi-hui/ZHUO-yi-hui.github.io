LEETCODE853 车队

发表于 2019-09-29

今天写一下几天前做的一道题吧
题目是车队
N 辆车沿着一条车道驶向位于 target 英里之外的共同目的地。
每辆车 i 以恒定的速度 speed[i] （英里/小时），从初始位置 position[i] （英里） 沿车道驶向目的地。一辆车追上另一辆车之后以相同的速度行驶。
注意：一辆车也可以是一个车队，即便一辆车在目的地才赶上了一个车队，它们仍然会被视作是同一个车队。

示例1：

输入：target = 12, position = [10,8,0,5,3], speed = [2,4,1,1,3]
输出：3

解释1:

从 10 和 8 开始的车会组成一个车队，它们在 12 处相遇。
从 0 处开始的车无法追上其它车，所以它自己就是一个车队。
从 5 和 3 开始的车会组成一个车队，它们在 6 处相遇。
请注意，在到达目的地之前没有其它车会遇到这些车队，所以答案是 3。

题解：

读完题目后，可以想到用时间的长短来判断是否追上前一辆车。当位置以降序排序时，即离终点距离近的车排在前面，如果后面的车到达终点时间小于前面车那么他们是一个车队，反之则新生成一个车队。那么我们就需要时间的排序和位置的排序顺序一样，那么我们需要一个类，里面有位置和时间两个属性。
注意：当车辆为0时，车队为0.
